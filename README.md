## Setup

### Frontend

**Installation**
```
cd frontend
npm install
```

**Setup**
```
npm run dev
```

### Backend

**Installation**
```
cd backend
pip3 install Flask
pip3 install Flask-SQLAlchemy
pip3 install flask-cors
```

**Setup**
```
python3 main.py
```
